# Daymark
 # Loga · 打卡记录

  > 一个轻量的目标打卡工具，记录每一步成长。无需注册，无需服务器，打开即用。

  ## 在线使用

  👉 [点击打开](https://innna327-source.github.io/Daymark/)

  ---

  ## 功能

  - ✅ 创建目标，设置总量与期待达成日
  - 📊 每次打卡自定义数量，自动累计进度
  - 🎉 达成目标时触发庆祝动画
  - 💾 数据备份导出 / 从文件恢复，换设备不丢失
  - 🎨 自定义列表背景（6 款渐变 + 自定义图片链接）
  - 📱 适配手机，可添加到主屏幕当 App 用

  ---

  ## 快速开始

  ### 方法一：直接使用（推荐）

  点击上方「在线使用」链接，无需任何安装。

  ### 方法二：本地运行

  ```bash
  git clone https://github.com/innna327-source/Daymark.git
  cd Daymark
  # 用浏览器直接打开 index.html 即可

  ---
  数据说明

  所有数据存储在浏览器本地（localStorage），不会上传到任何服务器。

  换设备 / 防丢失：

  1. 点击顶部 💾 按钮 → 导出备份文件（JSON 格式）
  2. 将文件保存到网盘或发送给自己
  3. 在新设备上打开页面 → 💾 → 导入备份文件
